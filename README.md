## Installation

1. Clone repository

2. Change to directory

```
cd vue-laravel-axios
```

3. Copy .env file


```
cp .env.example .env
```

4. Install dependencies

````
composer install
````
5. Modify `DB_*` value in `.env` with your database config.


6. Migrate
````
php artisan migrate
````

7. Install Node modules

````
npm install
````
8. Build

````
npm run prod
````
